# resource-cn-bb

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容相互独立，不针对任何特定域名或具体网站。仓库以资源归档为主要目的，方便后续查阅与分发。

## 项目简介

resource-cn-bb 是一个通用型 HTML 页面归档仓库。所有页面均为静态 HTML 文件，可直接在浏览器中打开查看。仓库结构清晰，页面按功能或主题分类存放，便于快速定位。

## 目录说明

```
├── pages/          # 存放所有归档的 HTML 页面
│   ├── type-a/     # 类型 A 的页面集合
│   ├── type-b/     # 类型 B 的页面集合
│   └── ...
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：每个子文件夹对应一类页面，文件夹名称反映页面主题。
- `assets/`：公共资源文件，页面可通过相对路径引用。
- 页面文件名建议使用英文小写加连字符，如 `example-page.html`。

## 页面归档说明

- 所有页面均为纯 HTML，不依赖后端服务。
- 页面之间无互相跳转或依赖关系。
- 归档内容不针对任何具体域名或网站，仅作为独立资源留存。
- 页面可能包含外部引用资源（如字体、图标库），但已尽量内联或本地化。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来补充或修正页面内容。
- 新增页面请放入对应分类文件夹，并确保文件命名规范。
- 不建议在页面中添加商业推广、诱导点击或恶意内容。
